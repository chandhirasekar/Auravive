# Product Images

This folder is where you should place your product images.

## Recommended Structure

```
public/images/products/
├── abc-malt-1.jpg
├── abc-malt-2.jpg
├── abc-malt-3.jpg
├── moringa-1.jpg
├── moringa-2.jpg
├── moringa-3.jpg
├── curry-leaves-1.jpg
├── curry-leaves-2.jpg
└── curry-leaves-3.jpg
```

## Image Guidelines

- **Format**: JPG, PNG, or WebP
- **Size**: 800x800px or larger (square aspect ratio)
- **Quality**: High resolution but optimized for web
- **File size**: Keep under 500KB per image

## How to Add Images

1. Take or create product images
2. Optimize them using tools like TinyPNG or Squoosh
3. Place them in this folder
4. The product data in `/data/products.js` already references these paths

## Current Status

The application currently uses placeholder images. Once you add real images here, they will automatically display in the product catalog and detail pages.
